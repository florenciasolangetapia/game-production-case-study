# La Última Tanda

### Game Production Case Study

*La Última Tanda* is a cozy narrative management game concept set in a small Buenos Aires bakery after closing time. The player prepares the final orders of the day while balancing limited ingredients, time, and the personal stories of recurring customers.

> This is a self-directed portfolio case study. The game and production scenario are fictional; the production approach and decisions are my own.

## Project Snapshot

| | |
|---|---|

## Production Challenge

The objective is to coordinate a small multidisciplinary team and deliver a polished vertical slice within a fixed schedule.

The production plan focuses on:

- Establishing a clear and testable core gameplay loop
- Controlling scope from pre-production onward
- Identifying technical and creative dependencies early
- Creating useful feedback cycles between disciplines
- Preparing the build for structured playtesting

## My Production Focus

As Producer / Project Manager, I am responsible for:

- Defining milestones and delivery criteria
- Turning the game concept into an actionable backlog
- Coordinating design, art, engineering, audio, and QA
- Tracking risks, dependencies, and scope changes
- Organizing playtesting and release-readiness activities
- Keeping project decisions visible and documented
